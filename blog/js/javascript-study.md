## DOM (문서 객체 모델, Document Object Model)

- 말 그대로 웹 페이지 내의 모든 컨텐츠를 객체로 나타내는 것이다
- 웹 페이지 = document
- document를 자유롭게 다루기 위해 객체화 하고자 구현된 개념이다
- HTML Tag, 글자, 속성 등 document에 담겨있는 모든 요소들을 하나 하나 객체화 한 단위를 가리켜 Node라고 한다
